CORE FEATURES

Four things Kept does.

Smart Capture and Claude Code Manager are free. Smart Recall and Knowledge Graph are Kept Pro, coming soon.

01

Smart Capture FREE

Type @kept in any chat window on ChatGPT, Claude, Gemini, Grok, or Kimi to save that conversation instantly. You can also let auto-sync run in the background — it checks for new conversations periodically and saves them without any action from you.

When you capture, you choose what to save: the full conversation, just the last few messages, or a specific branch. The Chrome extension handles deduplication automatically so you will never end up with duplicate copies of the same conversation.

For older conversations, use bulk import. Go to your AI platform's settings, download your data export, then drag and drop the file into Kept. Supported formats: ChatGPT data export (.zip), Claude export (.json), Gemini Takeout (.json), and generic Markdown (.md).

02

Smart Recall PRO

Smart Recall is a built-in chat inside the Kept desktop app. Type a question in plain English and it searches your full conversation archive to find the answer. Example: "What did I decide about authentication last month?" — Kept finds the relevant conversations and returns a cited answer.

Smart Recall uses semantic search, not just keyword matching. This means it finds relevant results even when you do not remember the exact words used. Results include the source conversation, platform, and date so you can go back to the original.

Smart Recall learns over time. Conversations you access frequently surface higher in results. Older facts that have been superseded by newer decisions gradually fade in relevance. This is a Pro feature. BYOK is supported: bring your own OpenAI, Anthropic, or OpenRouter key, or run local inference via Ollama.

03

Claude Code Manager FREE

The Claude Code Manager is a dedicated view inside the Kept desktop app for managing your Claude Code projects. It reads the local files that Claude Code writes to your machine — instructions, skills, and memory files — and shows them all in one place across every project you have ever worked on.

From the Claude Code Manager you can: browse all projects and their current instructions, search for a specific skill across all projects, copy a skill from one project and paste it into another, and save a set of skills as a reusable template for new projects.

This does not require any API key or cloud connection. It reads local files only. Nothing is uploaded anywhere.

04

Knowledge Graph PRO

The Knowledge Graph is an interactive 3D visualization of everything Kept has extracted from your conversations. After capturing conversations, Kept's knowledge engine parses them for entities: people, projects, technologies, decisions, and facts. Each entity becomes a node. Relationships between entities become edges.

Arcs are automatic groupings of related conversations into themes. Kept detects which conversations are about similar topics and groups them without any manual tagging required. Click any node in the graph to see the conversations it came from.

This is a Pro feature. It requires the desktop app to be running and at least 20-30 captured conversations before the graph becomes meaningfully populated.

TROUBLESHOOTING

Something not working?

Work through the relevant section below. If the issue persists, open a GitHub issue with your OS, browser version, and a description of what happened.

Steps:

  1. Make sure the Kept desktop app is open and running. The extension requires the app to be running to save conversations.
  2. Check that the extension is enabled. Go to chrome://extensions, find Kept, and confirm it is toggled on.
  3. Check extension permissions. The extension needs access to chat.openai.com, claude.ai, gemini.google.com, grok.com, and kimi.ai. Click the extension icon, open Settings, and confirm all platforms are enabled.
  4. Try a hard refresh on the AI platform tab (Cmd+Shift+R on Mac, Ctrl+Shift+R on Windows/Linux).
  5. If still not working, open the Chrome DevTools console on the AI platform tab and look for errors starting with [Kept]. Report any errors as a GitHub issue.

Steps:

  1. On Windows: right-click the .exe and choose "Run as administrator" once to grant permissions, then launch normally.
  2. On macOS: if you see "app is damaged" or "unidentified developer," open System Settings > Privacy & Security, scroll down, and click "Open Anyway" next to Kept.
  3. On Linux (AppImage): make sure the file is executable. Run: chmod +x Kept.AppImage then ./Kept.AppImage
  4. On Linux (.deb): install with sudo dpkg -i kept.deb then launch from your application menu or run kept from terminal.
  5. Check that you have at least 200MB of free disk space. Kept writes a local database and markdown vault.
